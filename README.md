RESUMEN:

Como dice en el nombre del repositorio este es un CRUD de tareas, es decir, podemos crear, leer, actualizar y eliminar las tareas.

en la carpeta /tasks van a encontrar lo siguiente:

En el archivo tasks.controllers.js - tenemos los controladores para hacer cada peticion.

En el archivo tasks.http.js - tenemos los servicios de cada peticion.

En el archivo tasks.router.js - tenemos las rutas de cada peticion.
